## 仓库说明
unity +  c++/nodejs 进行tcp通信，使用prototobuf序列化和反序列化
### 测试环境
- unity 5.3.4f1 (使用UGUI显示UI)
- c++ 11/nodejs6.3.1
- protobuf 2.6.x
- windows 7/10,linux-centos6.x
- IDE:visual studio in windows,eclipse for c++ in centos
